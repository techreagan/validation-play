# Rule Validation

Simple rule-validation API

## Routes

- HTTP [GET] "/" - Dev Details
- HTTP [POST] "/validate-rule" - Validation Rules

## Installation

Install all npm dependecies

```console
npm install
```

Install nodemon globally

```console
npm install -g nodemon
```

## Run Boilerplate

```console
node run dev
```
